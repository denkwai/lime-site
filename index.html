About

The name "Lime" stands for "LIME is a multiplatform engine".

It is a work-in-progress general puprose game and graphics engine.

Currently it is in the early development state with nightly source code snapshots provided.

Features

Cross-platform

  • Linux
  • Windows

Core

  • Multi-threading
  • Automatic memory and resource management

Renderer

  • Terrains
  • Particles
  • Post-processing
  • Lighting
    • Deferred Shading
    • Forward Light Indexed Deferred Rendering
    • Unlimited light count in both forward and deferred rendering
  • No fixed pipeline

Tools

  • Several mesh utilities
  • Scene editor
